### Instruções

1 - Exercício: Seleção de Estilo de Filme Favorito

Desenvolva um script em JavaScript utilizando SWITCH que permita ao usuário selecionar seu estilo de filme favorito a partir de uma lista pré-definida. O programa deve exibir uma lista numerada de opções de estilos de filmes e solicitar que o usuário insira o número correspondente ao seu estilo favorito.

As opções de estilos de filmes e seus respectivos números são os seguintes:

1 - Ação
2 - Comédia
3 - Drama
4 - Terror
5 - Ficção Científica
6 - Romance
O programa deve lidar com casos em que o usuário insira uma opção inválida, exibindo uma mensagem apropriada.

Após a seleção do usuário, o programa deve exibir a seguinte mensagem na página: "Seu estilo de filme favorito é: [estilo selecionado]".
